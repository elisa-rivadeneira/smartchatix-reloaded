# Documento 03 — Arquitectura Web

# Objetivo

La web de SmartChatics no tiene como objetivo informar.

Tiene como objetivo convertir visitantes en clientes.

Cada página deberá responder una pregunta específica del visitante y conducirlo al siguiente paso dentro del ecosistema.

---

# Objetivos estratégicos de la web

La web debe cumplir cinco objetivos:

1. Posicionar a SmartChatics como autoridad.
2. Generar confianza.
3. Convertir visitantes en alumnos.
4. Convertir alumnos en clientes empresariales.
5. Construir una comunidad alrededor de la marca.

---

# Los dos grandes ecosistemas

La web se divide en dos mundos perfectamente conectados.

# Estrategia SEO

Cada curso deberá tener:

Una Landing Page independiente.

Cada servicio empresarial deberá tener:

Una Landing Page independiente.

Cada problema del cliente deberá tener:

Un artículo específico.

Ejemplos:

- Cómo ahorrar tiempo con IA.

- IA para Excel.

- IA para docentes.

- Automatizar procesos.

- Dashboards con IA.

- IA para abogados.

- IA para contadores.

- IA para RRHH.

Cada artículo deberá enlazar naturalmente con un curso o servicio.

# Regla de diseño

Nunca mostrar primero la herramienta.

Siempre mostrar primero el beneficio.

Incorrecto:

"Automatizamos procesos con IA."

Correcto:

"Recupera horas de trabajo automatizando tareas repetitivas."

---

# Principio Estratégico

La web de SmartChatics no vende inteligencia artificial.

La web vende una transformación profesional.

La inteligencia artificial es simplemente el vehículo que hace posible esa transformación.

---

# Regla para todo el sitio

Cada sección de la web debe responder una sola pregunta del visitante.

Si una sección intenta responder varias preguntas al mismo tiempo, deberá dividirse.

La claridad siempre tendrá prioridad sobre la cantidad de información.
