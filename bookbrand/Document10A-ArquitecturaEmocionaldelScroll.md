# Documento 10A — Arquitectura Emocional del Scroll

# Introducción

Una página web no es una colección de secciones.

Es una historia.

Cada scroll representa un capítulo.

Cada capítulo despierta una emoción diferente.

Cuando las emociones aparecen en el orden correcto, la decisión de continuar surge de forma natural.

---

# Regla Fundamental

Nunca diseñar pensando en componentes.

Siempre diseñar pensando en emociones.

---

# El Viaje Emocional

## Escena 1

### Objetivo emocional

Curiosidad.

El visitante debe detenerse.

Debe sentir:

> "Esto no parece otra academia."

---

### Qué mostrar

Muy poco texto.

Una gran idea.

Mucho espacio.

Movimiento elegante.

Una frase poderosa.

Nunca una lista de cursos.

Nunca muchas tarjetas.

---

## Escena 2

### Objetivo emocional

Identificación.

Ahora el visitante piensa:

> "Eso también me pasa."

---

Mostrar:

Problemas cotidianos.

Tiempo perdido.

Procesos repetitivos.

Estrés.

Pero sin dramatizar.

---

## Escena 3

### Objetivo emocional

Esperanza.

El visitante descubre que existe otra forma de trabajar.

Aquí nace la curiosidad.

Todavía no aparece el curso.

Todavía no aparece el precio.

Solo aparece la posibilidad.

---

## Escena 4

### Objetivo emocional

Inspiración.

Aquí aparecen personas.

Historias.

Transformaciones.

No herramientas.

Historias.

---

## Escena 5

### Objetivo emocional

Confianza.

Ahora sí.

Cursos.

Empresas.

Servicios.

Porque el visitante ya comprende quiénes somos.

No antes.

---

## Escena 6

### Objetivo emocional

Decisión.

La llamada a la acción debe sentirse natural.

Nunca forzada.

No vender.

Invitar.

---

## Escena 7

### Objetivo emocional

Tranquilidad.

El usuario termina pensando

> "Aquí quiero aprender."

No

> "Aquí quieren venderme."

---

# Ritmo

Cada pantalla debe tener un ritmo diferente.

No repetir estructuras.

No repetir tamaños.

No repetir fondos.

El cerebro necesita novedad para mantener la atención.

---

# Principio Cinematográfico

Una película alterna:

Silencio.

Movimiento.

Primer plano.

Plano general.

Pausa.

Impacto.

La web debe hacer exactamente lo mismo.

---

# El uso del espacio

Después de una sección intensa...

Debe existir una pausa.

Después de mucho texto...

Debe aparecer una fotografía.

Después de una fotografía...

Debe aparecer una frase.

Después de una frase...

Una acción.

El ritmo es tan importante como el contenido.

---

# Microemociones

Cada interacción debe provocar una pequeña satisfacción.

Hover.

Scroll.

Animación.

Cambio de color.

Todo debe sentirse natural.

Nunca espectacular.

---

# Regla del 80%

Nunca mostrar todo.

Siempre dejar espacio para descubrir.

La curiosidad mantiene viva la atención.

---

# Lo que nunca haremos

Nunca colocar diez tarjetas seguidas.

Nunca mostrar ocho botones.

Nunca crear una pantalla completamente llena.

Nunca repetir la misma estructura cinco veces.

Nunca hacer que toda la página tenga el mismo peso visual.

---

# Lo que siempre haremos

Alternar.

Respirar.

Sorprender.

Inspirar.

Guiar.

---

# La última emoción

Cuando alguien cierre la pestaña debe quedarse pensando:

> "No vi solamente una página web."

> "Descubrí una nueva forma de entender el trabajo."

---

# Principio Estratégico

Una buena web informa.

Una excelente web inspira.

SmartChatix siempre diseñará experiencias que permanezcan en la memoria mucho después de que el visitante haya abandonado la página.
