# Documento 10A — Cinematic Design Language

# Introducción

SmartChatix no diseña páginas.

Diseña experiencias.

Cada página debe sentirse como el recorrido de una historia.

Cada scroll representa un nuevo capítulo.

Cada sección responde una pregunta.

Cada transición prepara emocionalmente la siguiente.

El visitante nunca debe sentir que navega una web.

Debe sentir que descubre una nueva forma de trabajar.

---

# Filosofía

El diseño debe desaparecer.

La historia debe permanecer.

Cuando alguien abandona SmartChatix no debería recordar las tarjetas.

Debería recordar cómo se sintió.

---

# El visitante es el protagonista

La empresa no es el héroe.

Los cursos no son el héroe.

La inteligencia artificial tampoco.

El protagonista siempre será la persona.

Todo el recorrido visual debe construirse alrededor de ella.

---

# La web como una película

Toda página debe seguir una estructura narrativa.

## Acto I

Curiosidad

↓

## Acto II

Identificación

↓

## Acto III

Esperanza

↓

## Acto IV

Confianza

↓

## Acto V

Acción

Nunca comenzar mostrando productos.

Nunca comenzar hablando de SmartChatix.

Siempre comenzar hablando del visitante.

---

# Ritmo Visual

Toda buena película alterna intensidad y calma.

La web debe hacer exactamente lo mismo.

Después de una sección intensa...

Debe existir una pausa.

Después de mucho texto...

Debe aparecer una imagen.

Después de una imagen...

Debe aparecer una idea.

Después de una idea...

Una invitación.

Nunca mantener el mismo ritmo durante toda la página.

---

# El Silencio Visual

El espacio vacío comunica.

Las pausas permiten pensar.

No llenar la pantalla.

No intentar explicar todo.

No competir por la atención.

La claridad siempre gana.

---

# Un protagonista por pantalla

Cada pantalla comunica una sola idea.

No cinco.

No siete.

No diez.

Si todo quiere llamar la atención...

Nada la consigue.

---

# La fotografía

Las imágenes no decoran.

Narran.

Queremos fotografías donde las personas estén:

Pensando.

Conversando.

Aprendiendo.

Creando.

Tomando decisiones.

No posando.

No señalando pantallas.

No mirando directamente a cámara.

---

# La luz

La luz simboliza comprensión.

Los espacios luminosos representan claridad.

Los fondos oscuros representan reflexión.

Nunca utilizar el contraste únicamente por estética.

Debe apoyar la narrativa.

---

# El movimiento

El movimiento acompaña.

Nunca protagoniza.

Las animaciones deben sentirse como una respiración.

Naturales.

Elegantes.

Lentas.

Nunca exageradas.

Nunca distractoras.

---

# La velocidad

Toda transición debe transmitir serenidad.

Preferimos movimientos suaves.

No efectos espectaculares.

No rebotes.

No explosiones.

No giros innecesarios.

---

# El Scroll

El scroll no es un desplazamiento.

Es una conversación.

Cada vez que el usuario baja...

Debe descubrir algo nuevo.

Nunca repetir la misma estructura durante varias secciones consecutivas.

---

# Alternancia

Evitar monotonía.

Alternar:

Texto.

Fotografía.

Espacio.

Color.

Movimiento.

Oscuridad.

Luz.

El cerebro disfruta la variedad.

---

# Jerarquía

Siempre debe existir un protagonista visual.

Puede ser:

Una frase.

Una fotografía.

Un video.

Una ilustración.

Nunca varios al mismo tiempo.

---

# Microinteracciones

Cada interacción debe producir satisfacción.

Hover.

Focus.

Scroll.

Cambio de color.

Todo debe sentirse orgánico.

Nunca exagerado.

---

# La emoción correcta

Cada parte del recorrido debe provocar una emoción específica.

Inicio

Curiosidad.

↓

Problema

Identificación.

↓

Historia

Esperanza.

↓

Cursos

Confianza.

↓

CTA

Motivación.

↓

Footer

Tranquilidad.

---

# El final

La última pantalla debe dejar una sensación.

No un mensaje.

Una sensación.

La persona debe cerrar la pestaña pensando:

"Aquí entendieron exactamente lo que necesito."

---

# Lo que nunca haremos

Nunca crear una web basada únicamente en componentes.

Nunca llenar la pantalla de tarjetas.

Nunca utilizar el movimiento como espectáculo.

Nunca construir una página donde todas las secciones tengan el mismo peso visual.

Nunca convertir la navegación en una sucesión de bloques repetitivos.

---

# Lo que siempre haremos

Contar historias.

Inspirar.

Respirar.

Guiar.

Sorprender con elegancia.

Invitar a continuar.

---

# Referencias cinematográficas

Nos inspira la capacidad narrativa de los grandes directores para controlar el ritmo, el silencio y la emoción.

No buscamos copiar un estilo visual.

Buscamos aprender cómo dirigir la atención del espectador mediante la composición, la luz y el tiempo.

---

# Principio Estratégico

Cada página de SmartChatix debe sentirse menos como un sitio web y más como una experiencia cuidadosamente dirigida.

El visitante no debe recordar el diseño.

Debe recordar cómo cambió su forma de pensar.
