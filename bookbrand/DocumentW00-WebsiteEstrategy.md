# Objetivo principal

La web de SmartChatix no existe para mostrar tecnología.

Existe para ayudar a las personas a descubrir una nueva forma de trabajar.

Toda decisión de diseño deberá responder esta pregunta:

> ¿Esto ayuda al visitante a comprender cómo puede trabajar mejor gracias a la tecnología?

---

# Objetivo Comercial

La web tiene cuatro objetivos.

1. Inspirar.

2. Generar confianza.

3. Conseguir alumnos.

4. Conseguir clientes empresariales.

# Qué nunca debe parecer

❌ Una empresa de software.

❌ Una agencia tecnológica.

❌ Una empresa de IA.

---

# Qué debe parecer

✅ Una empresa que ayuda a las personas a evolucionar profesionalmente.

# Qué debe sentir un visitante

Primeros 5 segundos

"Esto también es para mí."

30 segundos

"Entienden mi problema."

2 minutos

"Quiero aprender con ellos."

5 minutos

"Confío en esta empresa."

---

# Regla de Oro

Nunca comenzar una página hablando de herramientas.

Siempre comenzar hablando del problema del usuario.
